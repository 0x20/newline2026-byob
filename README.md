# Newline 2026 - BYOB Artwork

An interactive 3D artwork featuring iridescent bubbles and Frutiger Aero aesthetics, created by [Hackerspace Gent](https://hackerspace.gent) for the Bring Your Own Beamer event.

**Live Demo:** [newline.mateovandamme.com](http://newline.mateovandamme.com/)

## About

This piece promotes **Newline 2026**, a show and tell event for hackers, makers, and tinkerers of all shapes and kinds, taking place May 29-30-31, 2026 in Gent.

## Technical Details

Built with:
- [Three.js](https://threejs.org/) - 3D graphics library
- WebGL for hardware-accelerated rendering
- MeshPhysicalMaterial with transmission and iridescence effects
- Perlin noise terrain generation

### Features
- 25 iridescent bubbles with realistic physics
- Dynamic lighting with multiple colored point lights
- Procedurally generated terrain
- FPS counter (toggle with 'd' key)
